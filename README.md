# Enhanced Chat Notifications

Customizable screen and text notifications based on chat messages.

## Features

- **Custom notifications**: Configure up to 10 unique notifications based on game chat messages. Below an example with a purple screen + Antifire text based on the chat message stating that the antifire is about to expire.

- **Text Overlay**: Optionally display a text box (e.g. "Antifire") besides the screen notification.

![Custom notification example](https://i.imgur.com/zKpygjP.png)

## Use case examples, fully customizable
- Goading potion expiring → red flashing screen
- Receiving a valuable drop excluding runite ores → Gold solid screen
- Prayer regeneration potion has won off → Blue screen + text 'Prayer regen'
- Any game chat trigger → your own custom style
